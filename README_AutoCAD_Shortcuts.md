## 👤 用户配置系统使用指南

### 场景1：保存自己的快捷键配置

1. 在 AutoCAD 中设置好你的自定义快捷键
2. 打开工具 → 选择 **3. 用户配置管理**
3. 选择 **2. 创建新配置**
4. 输入配置名称（如：张三的配置）
5. 工具会自动保存当前所有自定义快捷键

### 场景2：切换到另一台电脑使用

**在原电脑：**
1. 工具 → 用户配置管理 → **6. 导出配置**
2. 选择你的配置，导出到 U 盘或云盘

**在新电脑：**
1. 复制工具文件到新电脑
2. 工具 → 用户配置管理 → **7. 导入配置**
3. 选择导出的 .json 文件
4. 选择 **3. 加载配置到当前 AutoCAD**
5. 重启 AutoCAD 即可使用

### 场景3：多人共用一台电脑

**用户A使用：**
1. 加载自己的配置（用户配置管理 → 加载配置）
2. 使用 AutoCAD

**用户B使用：**
1. 加载自己的配置
2. 使用 AutoCAD

**切换只需几秒钟！**

## 📋 配置管理详细功能

### 查看所有配置
显示所有已保存的用户配置，包括：
- 配置名称
- 描述
- 快捷键数量
- 最后修改时间

### 创建新配置
- 基于当前 AutoCAD 的自定义快捷键创建
- 可添加描述说明

### 加载配置
- 将保存的配置应用到当前 AutoCAD
- 会自动替换所有自定义快捷键
- **需要重启 AutoCAD 生效**

### 更新配置
- 将当前 AutoCAD 的快捷键保存到已有配置
- 用于更新自己的配置

### 删除配置
- 删除不再需要的配置

### 导出配置 ⭐
- 将配置导出为 .json 文件
- 可复制到其他电脑使用
- 可通过邮件/微信/云盘分享

### 导入配置 ⭐
- 从 .json 文件导入配置
- 支持导入其他电脑导出的配置

## 💡 使用技巧

### 快捷键命名建议
- 使用容易记忆的组合
- 避免与内置快捷键冲突
- 可以按功能分类（如：图层操作以 Q 开头）

### 推荐自定义快捷键

| 别名 | 命令 | 用途 |
|------|------|------|
| XX | EXPLODE | 分解 |
| ZZ | ZOOM E | 范围缩放 |
| QQ | LAYOFF | 关闭图层 |
| WW | LAYISO | 图层隔离 |
| EE | LAYTHW | 解冻所有图层 |
| AA | AREA | 计算面积 |
| DD | DIST | 测量距离 |

### 批量添加快捷键
在"管理快捷键"中选择"批量添加"，格式：
```
别名,命令,描述
XX,EXPLODE,分解
ZZ,ZOOM E,范围缩放
```

## ⚠️ 注意事项

1. **自动备份**：每次修改前会自动创建备份
2. **管理员权限**：修改 Program Files 下的文件需要管理员权限
3. **重启生效**：修改快捷键后需要重启 AutoCAD
4. **配置兼容**：导出的配置可在不同 AutoCAD 版本间使用

## 🆘 常见问题

**Q: 找不到 AutoCAD 安装？**
A: 可以手动输入 acad.pgp 文件路径，通常在：
```
C:\Program Files\Autodesk\AutoCAD 20XX\Support\acad.pgp
```

**Q: 修改后 AutoCAD 没有变化？**
A: 需要完全关闭并重新打开 AutoCAD

**Q: 如何备份所有配置？**
A: 复制整个配置目录：
```
C:\Users\[用户名]\.autocad_shortcuts\
```

**Q: 配置可以在 Mac 版 AutoCAD 使用吗？**
A: 配置文件是通用的，但路径需要手动指定
