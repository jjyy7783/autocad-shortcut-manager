# AutoCAD 快捷键管理工具 v2.3

## 开源项目 | 作者: jjyy7783

## 功能简介
AutoCAD 2004-2026 全版本快捷键管理工具，支持多用户配置方案切换，解决插件快捷键冲突。

## 核心功能
- ✅ 自动检测 AutoCAD 2004-2026 所有版本
- ✅ 导出/导入快捷键配置
- ✅ 多用户配置方案管理
- ✅ 智能命令选择器（中英文搜索）
- ✅ 330+ 常用命令数据库
- ✅ **解决插件快捷键冲突（天正等插件）**
- ✅ 自动加载脚本（启动时自动覆盖插件）
- ✅ 非破坏性修改（保留原始 PGP 文件）

## 使用方法
双击运行 `运行_快捷键管理工具.bat`

## 解决插件冲突
如果天正等插件覆盖了你的快捷键：
1. 运行工具，选择菜单 `6. 重新加载快捷键`
2. 选择 `2. 创建自动加载脚本`
3. 在 AutoCAD 中用 APPLOAD 将脚本添加到启动组
4. 以后每次启动都会自动加载你的快捷键
