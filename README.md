# AutoCAD 快捷键管理工具

AutoCAD 2004-2026 全版本快捷键管理工具，支持多用户配置方案切换、导入导出配置，并解决天正等插件导致的快捷键冲突问题。

## 功能特性

- 支持 AutoCAD 2004-2026 全版本自动检测
- 支持快捷键查看、添加、修改、删除
- 支持多用户配置方案管理
- 支持配置导出、导入、迁移到其他电脑
- 内置 330+ 常用命令数据库
- 支持中英文命令搜索的智能命令选择器
- 支持重新加载快捷键，解决插件覆盖问题
- 支持创建自动加载脚本，开机自动恢复快捷键
- 非破坏性修改，保留原始 PGP 文件

## 适用场景

- 想统一管理 AutoCAD 快捷键
- 多人共用一台电脑，需要快速切换配置
- 需要把自己的快捷键方案迁移到新电脑
- 被天正等插件覆盖快捷键，想自动恢复

## 快速开始

### 1. 启动工具

双击运行：

`运行_快捷键管理工具.bat`

### 2. 常见操作

- 查看当前快捷键
- 添加或修改自定义快捷键
- 创建个人配置方案
- 导出配置到其他电脑
- 生成自动加载脚本以对抗插件冲突

## 解决插件快捷键冲突

如果天正等插件覆盖了你的快捷键，可以这样处理：

1. 运行工具
2. 选择 `6. 重新加载快捷键`
3. 选择 `2. 创建自动加载脚本`
4. 在 AutoCAD 中使用 `APPLOAD` 将脚本添加到启动组
5. 以后每次启动 AutoCAD 时都会自动恢复你的快捷键

## 兼容性

- AutoCAD 2004-2026
- AutoCAD LT
- 多语言版本

如果自动检测失败，也可以手动指定 `acad.pgp` 文件路径。

## 配置存储

用户配置默认保存在：

`C:\Users\[用户名]\.autocad_shortcuts\`

每个配置文件为独立 JSON 文件，方便备份、分享和迁移。

## 使用提示

- 修改快捷键后，通常需要重启 AutoCAD 才会生效
- 修改前工具会自动备份原有配置
- 如果 AutoCAD 安装在受保护目录下，可能需要管理员权限
